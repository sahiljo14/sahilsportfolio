Projects

Mini Weather Station

Developed an advanced ESP8266-based weather monitoring system, integrating high-precision DHT22 and BMP180 sensors to deliver real-time temperature, humidity, and atmospheric pressure data. The project features an 16X2 LCD display for user-friendly visualization, a robust data-logging mechanism that displays the data on BLYNK app, and a low-power design optimized for prolonged outdoor use, showcasing my skills in IoT and embedded systems.

Mini Gaming Console

Built a compact NES emulator using an ESP32 microcontroller and an ILI9341 TFT LCD screen, capable of running classic NES games with optimized low-level code for smooth performance. The system was assembled on a general-purpose PCB with manually soldered components, demonstrating strong hands-on electronics skills. Powered by a rechargeable battery, the project showcases a blend of retro gaming and embedded systems design.
